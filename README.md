# EncodeDebugger
Translate text in to hash and base64
Package required:hashlib,hmac,tkinter,PIL,base64


Supported hash:
"MD5","SHA-1","SHA-224","SHA-256","SHA-384","SHA-512","Shake-128","Blake2b","Blake2s","MD5 with salt"
